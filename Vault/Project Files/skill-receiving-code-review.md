# skill-receiving-code-review

## מה הסקיל עושה
איך לטפל ב-code review feedback: לפני שמיישמים, לוודא הבנה ועומק טכני. מונע "ביצוע עיוור" ו"הסכמה ביצועית" כשהפידבק לא ברור או שגוי טכנית.

## מתי להפעיל
בקבלת פידבק ב-review, במיוחד אם הוא לא ברור או נראה שגוי.
