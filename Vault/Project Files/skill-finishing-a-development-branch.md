# skill-finishing-a-development-branch

## מה הסקיל עושה
מנחה את ההחלטה איך לסגור עבודה: merge, פתיחת PR, או cleanup. נכנס לפעולה כשהמימוש הושלם, הבדיקות עוברות, וצריך לבחור איך לאינטגרץ' את העבודה.

## מתי להפעיל
לאחר השלמת מימוש + verification.
