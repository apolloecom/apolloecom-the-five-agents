# skill-writing-plans

## מה הסקיל עושה
כתיבת תוכנית יישום מפורטת בעקבות מפרט/דרישות של משימה רב-שלבית. מנחה איך לפרק עבודה לצעדים ולקבוע checkpoints.

## מתי להפעיל
כשיש מפרט/דרישות והמשימה רב-שלבית, לפני שנוגעים בקוד.

## קבצים בתת-התיקייה
- **`SKILL.md`** — המסמך הראשי.
- **`plan-document-reviewer-prompt.md`** — פרומפט לסוכן שעובר על מסמך תוכנית ובודק אותה.

## למי הוא משויך
- **תפקיד:** שלב תכנון לפני ביצוע.
- **קוראים:** Claude Code לפני יישום.
