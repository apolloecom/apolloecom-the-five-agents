# root-env-files

## מה הקבצים עושים
- **`.env`** — קובץ משתני סביבה אמיתי המכיל מפתחות API ותצורת ריצה. מוסתר מ-git (ראה `.gitignore`).
- **`.env.example`** — תבנית פומבית עם placeholders של אותם משתנים, משמש כמסמך הוראות למפתחים שמשכפלים את הפרויקט.

## משתנים המוגדרים
- `ANTHROPIC_API_KEY` — מפתח API של Anthropic (placeholder ב-example).
- `ANTHROPIC_MODEL` — מודל ברירת מחדל (`claude-opus-4-7`).
- `APP_ENV` — סביבת הריצה (`development`).
- `LOG_LEVEL` — רמת לוגים (`info`).

## למי הם משויכים
- **תפקיד:** תצורת runtime של אפליקציות שיכתבו בעתיד תחת הפרויקט (סוכנים, סקריפטים).
- **בעלים:** מפתח הפרויקט.
- **קוראים:** קוד ה-runtime (טרם נכתב).

## מיקום
`/.env`, `/.env.example` (שורש)

## קבצים קשורים
- [[root-gitignore]] — שמסתיר את `.env` מ-git.
- [[root-claude-md]] — מתאר את ההקשר הפרויקט בו ה-API הזה ייעשה בו שימוש.
- [[skill-claude-api]] — הסקיל המתאים לעבודה עם ה-Anthropic SDK (כשיגיע הזמן לכתוב קוד).

## הערות אבטחה
- `.env` חסום ב-`.gitignore` — אסור לוודא commit שלו.
- גרסת ה-`.env.example` היא הקובץ היחיד מהשניים שעובר ל-git, ובאופן מכוון לא מכיל סודות אמיתיים.
