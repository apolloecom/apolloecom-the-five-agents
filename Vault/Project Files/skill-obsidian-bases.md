# skill-obsidian-bases

## מה הסקיל עושה
יצירה ועריכה של קבצי `.base` של Obsidian: views, filters, formulas, summaries. מאפשר לבנות תצוגות דמויות-מסד-נתונים מעל הערות הקיימות ב-vault.

## מתי להפעיל
בעבודה עם קבצי `.base`, או כשהמשתמש מזכיר Bases, table views, card views, filters או formulas ב-Obsidian.

## קבצים בתת-התיקייה
- **`SKILL.md`** — המסמך הראשי (קובץ יחיד).

## למי הוא משויך
- **תפקיד:** מבני נתונים מעל ה-vault.
- **קוראים:** Claude Code ביצירת views.

## מיקום
`/.claude/skills/obsidian-bases/`

## קבצים קשורים
- [[skill-obsidian-vault-workflow]] — ה-Bases יעבדו על קבצי הנושא ב-vault.
- [[skill-obsidian-markdown]] — תחביר frontmatter שעליו ה-Bases משאילים.
- [[obsidian-config]] — `bases: true` הופעל ב-`core-plugins.json`.

## הערות
ה-core plugin "bases" הופעל בתצורת ה-vault (`.obsidian/core-plugins.json`). ניתן להוסיף קבצי `.base` ב-vault לתצוגות מתקדמות של הנושאים שיתועדו.
