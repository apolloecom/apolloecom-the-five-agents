# root-claude-md

## מה הקובץ עושה
`CLAUDE.md` בשורש הפרויקט הוא קובץ ההוראות הראשי ל-Claude Code. הוא נטען אוטומטית בתחילת כל סשן ומכתיב ל-Claude את ההקשר של הפרויקט: תיאור הפרויקט ("The Five Agents"), מבנה התיקיות תחת `.claude/`, ומדיניות עבודה.

## למי הוא משויך
- **תפקיד:** קונפיגורציית פרויקט ברמה הגלובלית.
- **בעלים:** מפתח הפרויקט (Shachar Amar / apolloecom).
- **קוראים:** Claude Code בכל סשן (אוטומטי).

## מיקום
`/CLAUDE.md` (שורש)

- [[claude-folder-structure]] — תיקיית `.claude/` המוזכרת בקובץ.
- [[claude-settings]] — `.claude/settings.json` — הגדרות הפלאגינים שמופעלות יחד עם CLAUDE.md.
- [[skill-obsidian-vault-workflow]] — הסקיל שמתועד ב-CLAUDE.md ככזה שיש להפעיל בכל משימה.

## הערות
- עברית: ההוראות בקובץ כתובות בעברית, מתאר צוות סוכני AI לתוכן עם "מנכ"ל" ראשי.
- הקובץ מציין שפרטי הסוכנים יעודכנו בהמשך — נכון לעכשיו רק התשתית קיימת, לא הסוכנים עצמם.
