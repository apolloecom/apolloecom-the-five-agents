# skill-obsidian-markdown

## מה הסקיל עושה
יצירה ועריכה של Obsidian Flavored Markdown: wikilinks (`[[note]]`), embeds (`![[note]]`), callouts, properties (frontmatter), tags, ועוד תחביר ספציפי ל-Obsidian.

## מתי להפעיל
בעבודה עם קבצי `.md` ב-Obsidian, או כשהמשתמש מזכיר wikilinks, callouts, frontmatter, tags, embeds, או notes של Obsidian.

## קבצים בתת-התיקייה
- **`SKILL.md`** — המסמך הראשי (קובץ יחיד).

## למי הוא משויך
- **תפקיד:** תחביר Markdown ל-Obsidian.
- **קוראים:** Claude Code בעריכת תוכן ב-Vault.
