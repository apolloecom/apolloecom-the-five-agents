# skill-writing-skills

## מה הסקיל עושה
איך ליצור skills חדשים, לערוך קיימים ולוודא שהם פועלים לפני deployment. כולל best practices של Anthropic, עקרונות שכנוע ובדיקה עם תת-סוכנים.

## מתי להפעיל
ביצירה/עריכה של סקיל חדש, או לפני שמטמיעים סקיל בייצור.

## קבצים בתת-התיקייה
- **`SKILL.md`** — המסמך הראשי.
- **`anthropic-best-practices.md`** — practices מומלצים של Anthropic.
- **`persuasion-principles.md`** — עקרונות שכנוע לכתיבת skills יעילים.
- **`testing-skills-with-subagents.md`** — כיצד לבדוק skill עם subagents.
- **`graphviz-conventions.dot`** — מוסכמות ל-graphs ב-Graphviz.
- **`render-graphs.js`** — סקריפט לרינדור graphs.
- **`examples/CLAUDE_MD_TESTING.md`** — דוגמה לבדיקת CLAUDE.md.

## למי הוא משויך
- **תפקיד:** מטא-סקיל לכתיבת skills.
- **קוראים:** Claude Code בעת יצירת skill.
