# skill-requesting-code-review

## מה הסקיל עושה
מנחה איך לבקש code review בעת השלמת משימה, יישום פיצ'ר משמעותי או לפני merge. מאמת שהעבודה עומדת בדרישות.

## מתי להפעיל
בסיום משימות, יישום פיצ'רים, לפני merge.
